# Portfolio Sarah Martin - Data Analyst

Un portfolio professionnel développé avec Bootstrap, HTML5 et CSS3 avancé.

## Description

Ce portfolio présente les compétences et réalisations de Sarah Martin, data analyst senior. Il inclut des sections pour la présentation, le CV, le parcours académique, les expériences professionnelles, les compétences techniques organisées par catégories avec système d'accordion, les projets réalisés et les informations de contact.

## Fonctionnalités

- Design responsive avec Bootstrap 5
- Navigation fluide avec scroll smooth
- Animations au survol des cartes
- Timeline pour les expériences professionnelles
- **Système d'Accordion pour les compétences techniques** (basculement par catégories)
- 5 catégories de compétences : Langages, Visualisation, Bases de données, Outils, Expertise
- Barres de progression pour chaque compétence
- Formulaire de contact fonctionnel
- Images professionnelles via URLs externes
- Icônes Font Awesome pour une meilleure UX

## Technologies utilisées

- HTML5
- CSS3 (avec animations et transitions)
- Bootstrap 5
- JavaScript (ES6+)
- Font Awesome pour les icônes

## Installation et utilisation

1. Clonez ce repository
2. Ouvrez `index.html` dans votre navigateur web
3. Le portfolio est entièrement statique et ne nécessite pas de serveur

## Personnalisation

Pour adapter ce portfolio à vos besoins :

1. Modifiez les informations dans `index.html`
2. Ajustez les styles dans `assets/css/styles.css`
3. Ajoutez votre logique JavaScript dans `assets/js/script.js`
4. Remplacez les URLs d'images par vos propres photos
