## 🚀 UTILISATION

### **Pour Afficher**
1. Ouvrir `index.html` dans un navigateur
2. Page se charge avec animation 2.5s
3. Cliquer sur les liens de navigation
4. Changer de thème via icône palette
5. Soumettre formulaire contact

### **Pour Modifier**
- **Contenu** : Éditer index.html (sections)
- **Couleurs** : Éditer styles.css (--primary-color dans :root)
- **Animations** : Éditer styles.css (@keyframes)
- **Logique** : Éditer script.js (event listeners)

### **Pour Déployer**
- GitHub Pages
- Netlify (déploiement gratuit)
- Vercel (Next.js optional)
- Serveur web classique
